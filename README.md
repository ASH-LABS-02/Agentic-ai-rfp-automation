# Agentic AI – B2B RFP Automation

An Agentic AI system designed to automate the end-to-end B2B RFP response process for industrial manufacturing companies.

## Features
- Automated RFP discovery from websites and emails
- Intelligent RFP parsing (PDF/HTML)
- AI-driven SKU-to-spec matching with confidence scores
- Automated pricing and commercial quote generation
- Submission-ready RFP response packs

## Architecture
Sales Agent → Technical Agent → Pricing Agent → Master Agent

## Tech Stack
Python, FastAPI, Agentic AI, HTML mock websites
